How To Use

This service can easily view the data in the form of JSON.
Paste the JSON String into the box, and press the [Dump JSON] Button.
Immediately your JSON data is displayed in the form of easy-to-see.
Please use it for development, such as using JSON API.
Note: UTF-8 Charset Only.

System

This program runs without communication with the server.
Even sensitive information JSON can be safely visualized.
This site accesses GoogleAnalytics and GoogleAdsense, but does not use the information entered in the JSON.
