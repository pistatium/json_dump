How To Use

This service can easily view the data in the form of JSON.
このサービスはJSON形式のデータを見やすく表示します。
Paste the JSON String into the box, and press the [Dump JSON] Button.
上のテキストボックスにJSONを貼り付けて、[Dump JSON]ボタンを押せば
Immediately yout data is displayed in the form of easy-to-see .
即座に見やすい形で表示してくれます。
Extra things such as call-back will be removed automatically.
また、コールバックなど余計なものが付いていても自動で除去します。
Please use it for development, such as using the API.
APIを使った開発などにお使いください。
Note: UTF-8 Charset Only.

Tips

The green part becomes the course from a root.
緑色の部分はルートからのパスになってます。
If it is Javascript, You only just copy and paste it for your program.
Javascriptであればそのままコピペして使えます。
If you use other script language, it gose well in the same way.
他のスクリプト言語ユーザーでも同じように構造を辿っていけば使えると思います。
In the case of sequence, a child can furl when you click a parent-element.
配列の場合、親要素をクリックすると子が畳まれます。
When JSON data are big, please use it.
JSONデータが大きい場合にご利用ください。

System

This program is only written by Javascript.
このプログラムはJavascriptのみによって書かれています。
Because there is no need to send the data to an external server, you can use fast and safe.
外部のサーバーにJSONの情報を送る必要がないため、高速かつ安全に利用出来ます。
